# Test sets

This directory contains lightweight benchmark/test-set definitions used by the PepDesign manuscript experiments.

- `protein_level_test.csv`: protein-level held-out split for SOTA benchmarking.
- `family_level_test.csv`: family-level held-out split for SOTA benchmarking.
- `PPDbench_pep_fastas/`: PPDbench peptide FASTA shards used by the benchmark workflow.
- `split_summary.json`: summary of the protein-level and family-level split sizes.

Large receptor/peptide PDB structure directories, generated structures, docking outputs, and model checkpoints are not stored in GitHub.
